## What recipes are for

Recipes are reusable prompts, conventions, and habits that help faltoobot and faltoochat do better work.

## Good examples

### Release prep

Use faltoochat to:
- inspect modified files
- add review comments
- ask it to address those comments
- draft the release note

### WhatsApp triage

Use faltoobot to:
- summarize messages from a thread
- draft a clean reply
- rewrite it in a warmer or shorter tone

### Skill-based workflows

Store repeatable workflows as local skills so the tool can reuse them across sessions.

## Available recipes

## Principle

Keep recipes short, reusable, and easy to trigger. A good recipe should feel like a repeatable playbook, not a one-off prompt.
