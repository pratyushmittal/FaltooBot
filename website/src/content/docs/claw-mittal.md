## What it is

Claw Mittal is the WhatsApp-first member of the family. It lives on its own WhatsApp account and answers messages using your configured model.

## Current command

Today this guide maps to the existing `faltoobot` binary.

## Getting started

Install the current package:

```bash
uv tool install faltoobot
```

Configure it:

```bash
faltoobot configure
```

Pair the WhatsApp account once:

```bash
faltoobot auth
```

Run it in the background:

```bash
faltoobot install
```

Check it is running:

```bash
faltoobot status
```

## Useful commands

```text
/help
/reset
```

## Good fit

Use Claw Mittal when you want a private AI number for:
- drafting replies
- summarizing ideas on the go
- quick image-aware conversations
- lightweight day-to-day assistance from WhatsApp
