## Install the family

```bash
uv tool install mittals
```

Today the codebase still ships as `faltoobot` and `faltoochat`, but the Mittals family is the direction for the website and docs.

## The family

- **Claw Mittal** — the WhatsApp AI agent
- **Coder Mittal** — the terminal coding agent with review mode
- **Recipes** — reusable prompts and workflows that help both tools do better work
